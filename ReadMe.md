# README

> 简介: 转换markdown为html

> 找了好多markdown编辑器,但是没有找到一款可以通过命令行调用的,于是在网上找了一款解析器,写了这个东东.凑合着用吧.


## 编译

> 需要安装[git](http://msysgit.github.io/)与[mercurial](http://mercurial.selenic.com/downloads)


	./make.bat


## 使用方法

> 没有环境编译的可以直接下载exe文件使用

	./md2html.exe -f 文件名
